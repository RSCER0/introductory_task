Сущность publication используется для хранения статей, в ней имеется функциональный idtag, который устанавливает связь типа "многие-к-одному"
с таблицей tag.(много статей с одним тэгом)

Сущность tag описывает возможные функциональные тэги, имеет связь "один-ко-многим" с сущностью publication.(один тэг на много статей)

Сущность subsection описывает подразделы, имеет связь "многие-ко-многим" с сущностью publication.(много подразделов на много статей)

Сущность user описывает пользователей сайта, имеет связи "многие-ко-многим" с сущностью publication (много авторов на много статей) и "один-ко-многим" с сущностью comment (один пишет много комментариев).

Сущность comment используется для хранения комментариев, имеет функциональные ключ idpublication, который устанавливает связь "многие-к-одному" с сущностью publication "многие-к-одному" (много комментариев под одной статьей), и функциональный ключ iduser, который устанавливает связь "многие-к-одному" с сущностью user (один пользователь может написать несколько комментариев).

Сущность authorship - ассоциативная сущность для реализации связи "многие-ко-многим" сущности user с сущностью publication.

Сущность publication_subsection - ассоциативная сущность для реализации связи "многие-ко-многим" сущности subsection с сущностью publication.
